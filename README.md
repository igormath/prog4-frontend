# KIpetshop Frontend

Front-end do projeto da disciplina Programação 4 da Universidade de Pernambuco. Trata-se de um site de um Petshop.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
